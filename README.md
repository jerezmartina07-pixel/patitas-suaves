# 🐾 Patitas Suaves — Tienda Online

Tienda de accesorios para mascotas. Tecnología: HTML, CSS y JavaScript puro. Sin frameworks, sin dependencias, sin servidor.

## Estructura del proyecto

```
patitas-suaves/
├── index.html       ← Página principal
├── css/
│   └── style.css    ← Todos los estilos
├── js/
│   └── main.js      ← Lógica (carrito, filtros, productos)
└── README.md
```

## Personalización rápida

- **Cambiar número de WhatsApp:** Buscar `5491100000000` en `index.html` y `main.js` y reemplazarlo con tu número (formato internacional, sin + ni espacios).
- **Cambiar precios:** Editar el array `PRODUCTOS` en `js/main.js`.
- **Agregar productos:** Agregar un objeto nuevo al array `PRODUCTOS` con el mismo formato.
- **Cambiar colores:** Editar las variables CSS en `:root` al inicio de `css/style.css`.
